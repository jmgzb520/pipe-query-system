# 多品牌多联机管径查询系统 - 纯 HTML 版本

## ✅ 已完成转换

已成功将 Next.js 项目转换为独立的 HTML 版本，无需任何后端服务器即可运行！

## 📦 文件信息

- **文件名**: `pipe-query.html`
- **文件位置**: `public/pipe-query.html`
- **文件大小**: 33KB
- **访问地址**: `http://localhost:5000/pipe-query.html`

## 🌐 访问方式

### 方式一：通过项目访问（推荐）
```
http://localhost:5000/pipe-query.html
```

### 方式二：直接打开
下载 HTML 文件后，双击即可在浏览器中打开，无需任何服务器。

### 方式三：部署到任何静态网站托管服务
- GitHub Pages
- Netlify
- Vercel
- 任何支持静态文件的服务器

## 🎯 功能特性

### ✨ 完整功能
- ✅ 7个品牌查询（海信、日立、美的、大金、三菱重工、三菱重工海尔、格力）
- ✅ 125个型号选择
- ✅ 240条管径规格数据
- ✅ 级联选择（品牌 → 型号）
- ✅ 实时查询
- ✅ 美观的卡片式展示
- ✅ 响应式设计（支持手机、平板、电脑）

### 🎨 界面设计
- 使用 Tailwind CSS 进行样式设计
- 渐变背景和卡片效果
- 清晰的气管/液管标识（🔵/🟢）
- 流畅的交互体验

## 🚀 优势

### 相比 Next.js 版本的优势
1. **零依赖**：不需要 Node.js、npm、pnpm
2. **零服务器**：不需要后端服务器、数据库
3. **易部署**：可部署到任何静态网站
4. **可离线**：下载后可离线使用
5. **快速**：无网络请求，即时响应
6. **简单**：单个文件，便于分享

### 技术栈
- 纯 HTML5
- 原生 JavaScript
- Tailwind CSS (CDN)
- 无任何框架依赖
